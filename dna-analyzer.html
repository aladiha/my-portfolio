DNA SequencesAnalyzer

C++ Application

The Idea

Dna Analyzer would help you store and load different dna sequences, As well as make different operations on them like duplicating dna, Finding sub sequence in the sequence, Counting sub sequences, Make paired sequences, Slicing them and more.

Project Design

Command Design Pattern for more readable / efficient code, Every command would be handled in it's own class.

Bit manipulation for more efficient memory management.
